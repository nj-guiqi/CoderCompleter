# CoderCompleter

https://competition.nowcoder.com/exam/199/376

使用Deepseek-Coder-1.3B模型作为基础预训练模型。不限定使用的数据集、数据清洗、数据混合方案。

对基础代码模型进行Java场景的代码补全（FIM）能力准确率调优。
